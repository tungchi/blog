  • byte为1个字节,8位,首位为符号位,0为正,1为负数,所以范围是00000000~01111111,即0~127,以及11111111~10000000,即-1~-128,则byte的值范围为-128~127
  • 同理,int 4个字节,为32位,则范围为-2147483648~2147483647
  • short为2个字节,16位,long为8个字节,64位
  • char为两个字节,但是没有负数,所以范围是0~2^16-1
  • 重点讲下float和double,float是4个字节,16位,double是8个字节,64位
  • 先讲float,第一位表示符号,1为负数,0为正数,,然后有8位是指数位,剩下的是尾数位:
    指数位有符号位,则值有10000000~11111111(-0~-127),00000000~01111111(+0~+127),底数为2,则取值范围为2^127~2^(-127)
  • 以<<,>>为例子,byte为例子,<<左位移,低位补0,>>位移,高位补0
  • byte a=127,转化为二进制1111111,其实最前面还有一位,01111111,首位0为正数,1为负数
  • a<<1,表示a左移一位,则变成11111110,负数转化为十进制,要先取反再加1,再加上符号位,即:
  • 1取反:10000001 ;2加一:10000010 3转换:-2
  • 还是以byte a=127为例子,a>>1,即00111111,转化为正数,+63

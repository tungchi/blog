算法描述:

  • 需要解决的问题小且简单(即不适用大型的复杂的问题)
  • 将一个问题分成n个相同的小问题

汉诺塔问题:

  • 三座塔,A塔有64个盘子,最底部盘子最大,向上依次变小
  • 现在要求将A塔的盘子移到C塔,一次只能搬一个盘子,小盘子必须在大盘子上面
  • 问共需要搬盘子多少次?

解题思路:

  • 设有n个盘子,那么要将底部最底下的第n个盘子搬到C塔,需要把上面n-1个盘子先搬到B塔,再从B塔搬到C塔,
  • 要把第n-1个盘子搬走,需要把上面n-2个盘子全部搬走
  • 要把第n-2个盘子搬走,需要把上面n-3个盘子全部搬走
  • .......
  • 要把第3个盘子搬走,需要把上面2个盘子全部搬走
  • 要把第2个盘子搬走,需要把上面1个盘子全部搬走
  • 注意只有三座塔且小盘子在大盘子上面

解:

  • 令F(n)=2F(n-1)+1,F(n-1)=2F(n-2)+1`````F(2)=2F(1)+1,F(1)=1
  • 数学公式推演:
    2^0F(n) = 2^1F(n-1)+2^0
    2^1F(n-1)= 2^2F(n-2)+2^1
    2^2F(n-2)= 2^3F(n-3)+2^2
    ......
    2^(n-2)F(2)=2^(n-1)F(1)+2^(n-2)
    2^(n-1)F(1)=2^(n-1)
    全部相加,再左右等式相减
    F(n) = 2^0+2^1+2^2+......+2^(n-2)+2^(n-1)
    左右两边*2
    2F(n) = 2^1+2^2+2^3+......+2^(n-1)+2^n
    上面两个公式相减
    F(n) = 2^n - 2^0,
    则F(n) = 2^n -1 ;

    move_times = move_times+1;

    }else{
    move(src, temp, dst, number-1);
    move_times = move_times+1;
    System.out.println("move "+src+" ."+number+" to "+dst);
    move(temp, dst, src, number-1);
    }
    }
